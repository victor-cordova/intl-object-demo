Intl.Collator

Developer: Hey browser, is there some way I can sort some elements considering that each language has different alphabets?

Browser: Of course, use Intl.Collator. Here's a simple example where the letter √§ has a different order depending on the language

    Code:

    const sortedElements = elementsToSort.concat().sort(new Intl.Collator(locale).compare)

    Intl.DateTimeFormat

    Developer: Hey browser, there are so many ways to format dates, is there a way to format dates for a specific locale?

    Browser: As usual, there is. Just use Intl.DateTimeFormat and you're good to go. For example, let's format today's date:


    Code:

    const formattedDate = new Intl.DateTimeFormat(locale).format(new Date())

    Intl.ListFormat

    Developer: Hey browser, I'm making a cooking app and want to show a list of ingredients. As always, each locale has a different way to format list elements.Do you have for me to help form lists with connectors in them according to the locale?

    Browser: A simple API as always. Go with Intl.ListFormat, it supports conjunction, disjunction and unit formatting. Here's an example

      Code

      const formattedList = new Intl.ListFormat(locale, { style: 'long', type: 'conjunction' }).format(elementsForListFormat)

      Intl.NumberFormat

      Developer: Hey browser, I'd like to format numbers according to a locale and a format like money. I'm working on adding a payment screen. What to do?

      Browser: Worry not, just use Intl.NumberFormat and problem solved. I created a demo for you to try it out:

      Code

      const formattedCurrency = new Intl.NumberFormat(locale, { style: 'currency', currency, }).format(numberToFormat)

      Intl.PluralRules

      Developer: Hey browser, I'm creating a formatting function to display ordinal (or cardinal) numbers according to a locale for a ranking feature (1st, 2nd, 3rd). It's hard to tell which suffix I should use. Can you give me a hand?

      Browser: Yes and no I'd say. Here I can help you with some plural rules for you to create the suffix yourself so you'll probably want to create a map of plural rule -> suffix. Here's an example of what I mean by plural rule:

      Code

      const pluralRule = new Intl.PluralRules(locale, { type: 'ordinal' }).select(count)

      Intl.RelativeTimeFormat

      Developer: Hey browser, I'm still working on that online store and want to create a tracking screen. However, it's kind of hard to generate these "Your package arrives in X time". Is there a way to generate relative time periods according to a locale. A bit of help?

      Browser: I have exactly what you need. Just use Intl.RelativeTimeFormat, pass a value and a unit and voila! Here's a playground for you to see how it works

      The value formatted with the unit is:

      Code

      const formattedDate = new Intl.RelativeTimeFormat(locale, { style: 'narrow' }).format(value, unit)

      Intl.Locale

      Developer: Hey browser, I hate implementing functions to parse unicode locale identifiers (as we all do). Can't you just do this for me?

      Browser: Of course I can. Just use Intl.Locale, pass your unicode locale identifier and use whatever attributes you need. Here's how the selected locale looks like deconstructed

      Code

      const localeObj = new Intl.Locale(locale)
      const propertiesOfLocale = `Properties of selected locale are: baseName ${localeObj.baseName}, calendar ${localeObj.calendar} , language ${localeObj.language}, region ${localeObj.region}`

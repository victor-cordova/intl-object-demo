Intl.Collator

Sorts elements considering the locale. Surprisingly, some letters don't hold the same position in the alphabet in different languages

Example

This example demonstrates how the letter z actually comes before √§ in the Swedish language

    Code:

    const sortedElements = elementsToSort.concat().sort(new Intl.Collator(locale).compare)

    Intl.DateTimeFormat

    Format dates for a specific locale

    Example

    Let's format today's date:

    Code:

    const formattedDate = new Intl.DateTimeFormat(locale).format(new Date())

    Intl.ListFormat

    Form lists with connectors according to the locale

    Example

    Browser: A simple API as always. Go with Intl.ListFormat, it supports conjunction, disjunction and unit formatting. Here's an example

      Code

      const formattedList = new Intl.ListFormat(locale, {
  style: 'long',
  type: 'conjunction'
}).format(elementsForListFormat)

      Intl.NumberFormat

      Format numbers according to a locale and a format like money.

      Example

      This API lets you format lists using conjunctions, disjunctions and unit formatting

      Code

      const formattedCurrency = new Intl.NumberFormat(locale, {
    style: 'currency',
    currency,
}).format(numberToFormat)

      Intl.PluralRules

      Display ordinal (or cardinal) numbers according to a locale

      Example

      This api provides plural rules that you can use for adding plural suffixes. However, you need to do this rule -> suffix translation on your own since every language has different values.

      Code

      const pluralRule = new Intl.PluralRules(locale, {
  type: 'ordinal'
}).select(count)

      Intl.RelativeTimeFormat

      Generate relative time periods according to a locale.

      Example

      The value formatted with the unit is:

      Code

      const formattedDate = new Intl.RelativeTimeFormat(locale, {
  style: 'narrow'
}).format(value, unit)
